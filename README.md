# Artisanal Bakery (Astro + Tailwind)

## Requisitos
- Node.js 18+

## Pasos (VS Code)
1. Abre esta carpeta en VS Code.
2. En la terminal:
   - `npm install`
   - `npm run dev`
3. Abre el navegador en la URL que te muestre Astro (normalmente http://localhost:4321).

## Archivos importantes
- `src/pages/index.astro` (la landing)
- `src/styles/global.css` (tailwind + helpers)
- `public/scripts/animations.js` (ripple, smooth scroll, reveal, nav activo)
